# 🌦️ Clima X

Aplicação simples de previsão do tempo, feita com **HTML, CSS e JavaScript** consumindo a API do **OpenWeatherMap**.  
O usuário digita o nome de uma cidade e recebe informações em tempo real sobre o clima.

## 🚀 Funcionalidades

- Buscar clima de qualquer cidade 🌆  
- Exibir temperatura atual 🌡️  
- Mostrar descrição do tempo (nublado, ensolarado, etc.) ☁️  
- Ícone correspondente às condições climáticas 🌦️  
- Bandeira do país da cidade selecionada 🇧🇷  
- Umidade relativa do ar 💧  
- Velocidade do vento 🌬️  

## 🛠️ Tecnologias usadas

- **HTML**  
- **CSS**  
- **JavaScript**  
- [OpenWeatherMap API](https://openweathermap.org/api)  
- [FlagCDN](https://flagcdn.com/)
